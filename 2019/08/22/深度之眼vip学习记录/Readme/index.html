写在前面

2019.08.22 大果  热度 ℃

     这一系列是打算记录在报名深度之眼vip后的学习历程的博客，因为课有点多，一起学就乱了，而且找些重要的东西每次都要从头捋真的太难受了…（他们那个打卡翻起来真的好慢啊….
     不过总体来讲，我还是挺喜欢的，深度之眼是一家AI教育的公司，和市面上许多自己讲课的公司不同，他们利用了现有的公开课（比如吴恩达的Machine Learning，李飞飞的CS231n等等），在此基础上再带领大家去做课程里的作业，站在巨人的肩膀上，既节约成本又避免市场上各种教育机构讲课能力参差不齐的情况，还解决了大家在看公开课没有配套习题的烦恼，比如对吴恩达先生的ML课程，因为年代久远用的编程语言还是Octave，在Python当道的现在，去学Octave总有点不甘不愿，但达叔的课又非常经典简直入门必看，所以深度之眼的老师把原作业改成了Python版，有没有借鉴别人的我不知道，反正对我来说挺好的，省不少时间。
     值得吐槽的是他们的打卡模式，对于我这种状态型选手，每天做重复一样的打卡，实在做不到呀，状态来了又只能做一天的卡简直要疯，而且有的卡又非常简单，有的卡可能一天内根本解决不了，所以我报了给vip，随时随地看，但是这样吧，看到哪又忘了，所以想在这里记录一下，纯粹就是为了记录一下进度，毕竟课真的有点多，学起来就乱了= =
     如果今后能对谁有帮助，那自然好，但是这不是我的初心，毕竟正儿八经的写博客分享每天所学，真的好累啊…我可能会学不完就放弃治疗了，所以真的只是随意记录一下…嗯
